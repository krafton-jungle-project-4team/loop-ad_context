# Documentation Style Rules

## RULE-DOC-001

README.md는 사람용 소개와 운영 방법을 담당하고, AGENTS.md는 AI 진입점을 담당한다.

## RULE-DOC-002

service context는 단일 서비스 책임을 설명하고, workflow context는 여러 서비스가 엮인 흐름을 설명한다.

## RULE-DOC-003

rules 문서는 엄격한 schema가 아니라 반복 인용할 느슨한 문구 묶음으로 관리한다.

## RULE-DOC-004

평가 결과는 사용자가 명시하지 않으면 파일로 생성하지 않고 대화 리포트로만 제공한다.
