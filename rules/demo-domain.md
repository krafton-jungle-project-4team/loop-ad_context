# Demo Domain Rules

이 문서는 demo frontend의 공통 도메인 표현에 관한 느슨한 문구 묶음이다. frontend 구현
세부사항의 원본으로 쓰지 않는다.

## RULE-DEMO-001

demo repository 이름과 public domain은 `demo-shoppingmall` 호환을 유지할 수 있다. 다만 화면,
fixture, event, 사용자 시나리오는 호텔 예약 데모로 제공한다.

## RULE-DEMO-002

데모 화면은 호텔 검색, 호텔 상세 조회, 예약 시작, 예약 완료, 예약 취소, 프로모션 클릭/랜딩
흐름을 보여준다. 쇼핑몰 상품/장바구니/구매 흐름을 새 기준 시나리오로 설명하지 않는다.
